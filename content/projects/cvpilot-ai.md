Built a portfolio-grade SaaS product for resume analysis and job-description matching.

The application includes an authenticated dashboard, PDF/DOCX/TXT resume parsing, manual CV input, AI-powered structured analysis, ATS score breakdowns, keyword coverage, optimized CV generation, charts, analysis history, and profile management.

The stack combines Next.js 15 App Router, TypeScript, Tailwind CSS, shadcn/ui, Clerk, PostgreSQL, Prisma, OpenAI, Zod, React Hook Form, Zustand, Recharts, Docker, and GitHub Actions.

CVPilot AI also includes deterministic local heuristics as a fallback when OpenAI quota is unavailable, keeping the product demoable and resilient during development.
