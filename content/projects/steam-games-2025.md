Academic data analysis project focused on games published on Steam, using the Steam Games 2025 dataset from Kaggle to explore genre performance, estimated users, playtime, and review patterns.

The analysis studies which genres concentrate more estimated users per game, which ones show higher average playtime, how free and paid games differ, and whether positive ratings are related to popularity or retention.

The project was built with Python, pandas, matplotlib, Jupyter Notebook, and supporting PowerShell scripts to reproduce the full workflow from preprocessing to figures and executed notebooks.

The repository also includes the final report, presentation material, and publication assets prepared for sharing the project across GitHub and professional platforms.
